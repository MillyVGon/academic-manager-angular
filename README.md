# AcadManager – Sistema de Gerenciamento Acadêmico

O AcadManager é uma aplicação web desenvolvida em Angular, criada com o objetivo de gerenciar projetos acadêmicos de forma organizada, intuitiva e eficiente. A aplicação permite criar, listar, visualizar e gerenciar projetos, servindo como base prática para o estudo do framework.

Este projeto foi desenvolvido com o objetivo de praticar e aprender conceitos fundamentais do Angular, focada em boas práticas de desenvolvimento front-end.

A aplicação explora a organização do código em módulos e funcionalidades, utilizando recursos como componentes standalone, services, signals, rotas e resolvers. Além de adotar uma estrutura bem definida, com separação clara entre serviços, interfaces e responsabilidades, facilitando a manutenção, leitura e evolução do projeto.
